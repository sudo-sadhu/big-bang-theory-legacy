# tbbt-fan-page-legacy

An immersive, nostalgic static fan website dedicated to CBS's television series *The Big Bang Theory*. This project presents interactive directories of episodes, cast profiles, iconic moments, and behind-the-scenes trivia using HTML, CSS, and Vanilla JavaScript.

## Tech Stack
* **Markup:** HTML5 (semantic layout structure)
* **Styling:** Custom CSS3 with dynamic variables and keyframe animations
* **Logic:** Vanilla JavaScript (ES6)
* **Data Source:** Integrates the TVMaze API for real-time cast records and episode graphics

## Key Features
* 🌌 **Symmetric Starry Background**: CSS-driven glowing ambient background.
* 🎬 **Episode Database**: Full 12-season directory sorted by season or IMDb rating with instant search.
* 👥 **Cast & Crew Directory**: Profiles featuring quotes and live cast photographs sourced dynamically.
* 📣 **Bazinga! Button**: An interactive tribute toast effect.

## Local Setup
1. Clone this repository:
   ```bash
   git clone git@github.com:sudo-sadhu/big-bang-theory-legacy.git
   ```
2. Open `index.html` directly in your browser or run a local static server:
   ```bash
   npx serve .
   ```
